📊 CEO Economics — Análise de Relatórios Financeiros
📑 DRE (Demonstração de Resultado)
  • Receita Líquida → Lucro Bruto → EBITDA → Lucro Líquido
  • Margem bruta ideal: >40% (tech), >20% (varejo)
  • EBITDA/Receita: eficiência operacional
  • Crescimento YoY: sinal de saúde do negócio
⚖️ Balanço Patrimonial
  • Ativo Circulante vs Não Circulante
  • Dívida Líquida / EBITDA: <2x = saudável
  • ROE (Retorno sobre PL): >15% = bom
  • Liquidez Corrente: >1.5 = seguro
💸 Fluxo de Caixa
  • FCO (Operacional) > Lucro Líquido = qualidade
  • CAPEX/Receita: intensidade de capital
  • Free Cash Flow = FCO - CAPEX
  • FCF Yield = FCF / Market Cap
📝 Proposta Executiva
  • Problema → Solução → Impacto → ROI
  • Executive Summary: máximo 1 página
  • Dados quantitativos sempre
  • Call to action claro e específico

Linux terminal (comandos básicos, permissões, scripts). Redes: TCP/IP, DNS, HTTP/S, portas. Python básico: variáveis, loops, funções, requests.

📊 CEO Economics — Economista Reativada

// Para quem sabe mas esqueceu. Conhecimentos poderosos para a vida.

📑 Análise de DRE
  • Receita Líquida → Lucro Bruto → EBITDA → Lucro Líquido
  • Margem EBITDA: >20% = eficiente, >40% = excelente
  • Crescimento de receita YoY: >15% = saudável
  • COGS/Receita: quanto custa produzir
  • SG&A/Receita: eficiência administrativa
⚖️ Balanço Patrimonial
  • Ativo = Passivo + Patrimônio Líquido
  • Dívida Líquida/EBITDA: <2x = saudável, >4x = risco
  • ROE = Lucro Líquido / PL: >15% = bom
  • Liquidez Corrente = AC/PC: >1.5 = seguro
  • Giro de Ativos: eficiência de uso dos ativos
💸 Valuation Básico
  • P/L (Preço/Lucro): <15 = barato, >30 = caro
  • EV/EBITDA: valor da empresa vs geração de caixa
  • DCF: valor presente dos fluxos futuros
  • P/VP: preço vs valor patrimonial
  • Dividend Yield: retorno em dividendos
📝 Proposta Executiva Poderosa
  • 1. Problema: Qual dor você resolve? (1 parágrafo)
  • 2. Solução: Como você resolve? (específico)
  • 3. ROI: Quanto retorna? (números reais)
  • 4. Prova: Cases, dados, credibilidade
  • 5. CTA: Próximo passo claro e urgente
🎯 Tomada de Decisão Estratégica
  • Matriz SWOT: forças, fraquezas, oportunidades, ameaças
  • Análise custo-benefício quantificada
  • Decisão de 10/10/10: como me sentirei em 10 min, 10 meses, 10 anos?
  • Pré-mortem: o que pode dar errado?
  • Regra de 2 minutos vs decisões importantes

  • Krashen: adquirimos idiomas por INPUT, não por estudo
  • Input compreensível (i+1): 95% entendido + 5% novo
  • Nunca traduzir mentalmente — pensar no idioma
  • Gramática emerge naturalmente com exposição
  • Fluência = horas de input qualificado
🃏 Anki SRS — Vocabulário

  • Formato: imagem + áudio + frase contextual
  • Revisão: 10–15 min/dia (automático)
